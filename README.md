# Sistema-de-Gest-o-de-Campeonatos-de-Futebol
    O projecto consiste no desenvolvimento de um sistema de gestão de campeonatos de futebol, capaz de gerar tabelas de jogos, apresentar classificações e informações das equipas. O objectivo é facilitar a organização das competições e aplicar conhecimentos adquiridos na disciplina de Programação 2.
